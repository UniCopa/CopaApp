Creates the ListView with SingleEvents in user defined colors (black as default).

In the ListView the time, the date and buttons are shown. It also handles the clicks on the buttons. If the "change"-button, which is only shown if the user gots the right to change, is clicked the ChangeSingleEventActivity starts. So the user can fill in his changes. If the "details"-button is clicked the SingleEventActivity starts and the user can see all details.
